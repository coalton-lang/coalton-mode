What's here:

### clj

A small clojure project, for protocol and UI comparison. To run,
install the lsp server:

``` sh
brew install clojure-lsp
```

Then open `clj/src/main.clj` and enable eglot mode in emacs:

``` emacs
M-x eglot
```
